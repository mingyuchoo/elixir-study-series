# 블로그 사이트

## 000

Elixir로 된 블로그 사이트를 제작할거야. 홈페이지와 상세 페이지가 있어야 돼.

### 공통 구조

- 헤더 네비게이션
- Footer

### 홈페이지 구조

- Carosel Hero 섹션 (인기 블로그 글)
- 인기 블로그 Grid
- 카테고리별 블로그 Grid
- 이메일 구독하기 CTA

### 상세 페이지 구조

- 제목
- 메타데이터 (저자, 태그, 읽는데 걸리는 시간)
- 썸네일
- 내용
- ToC

### 블로그 필수 정보

- 저자
- 태그
- 썸네일 이밎
- 내용 (Markdown)
- 요약
- 읽는데 걸리는 시간

샘플 블로그 포스트는 50개 정도 제작해주고 블로그 글은 마크다운으로 관리할거야.
